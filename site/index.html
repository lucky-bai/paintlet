MS Paint-style editor for macOS

Windows Paint,
in macOS clothing.

A small, light raster editor that borrows Windows 11 Paint's layout and interactions — then renders them in macOS clothing: a native transparent title bar, SF Pro, system controls, and full dark mode. A Windows Paint user should recognize it in five seconds and still feel it belongs on their Mac.

Download for macOS View on GitHub

Universal build (Apple Silicon + Intel) · signed & notarized · macOS 10.15+

Everything you'd reach for

The full Paint tool set — no layers, no alpha, no AI. Small on purpose.

Freehand

Pencil, brush, and eraser with a continuous width slider; left button paints Color 1, right paints Color 2.

Shapes

Line, curve, rectangle, rounded rectangle, ellipse, and polygon at four widths. Shift constrains to 45°/square/circle.

Fill & eyedropper

Leak-tight flood fill, plus a color picker that shows the sampled color under the cursor.

Text

Multi-line, with a live font preview, size steppers, and bold/italic/underline/strikethrough; reposition before committing.

Selection

Rectangular marquee or free-form lasso, with move, eight-grip resize, and transparent backgrounds — never a solid block.

Clipboard

Copy / cut / paste (⌘C / ⌘X / ⌘V) through the system clipboard.

Save / Open

One-step save as PNG or JPEG; the title bar tracks the file and any unsaved changes.

Image ops

Resize, crop, flip, rotate, and drag any canvas edge to crop or extend — all undoable.

Zoom & pan

0.25×–8× crisp pixelated scaling, fit-to-window, pinch / ⌘-wheel zoom, and space-drag panning.

Undo / redo

⌘Z / ⇧⌘Z across every edit, including resizes and crops.

Native macOS UI

A real menu bar and shortcuts, a Win11-style ribbon, an in-app color picker, the MS Paint palette, and a live status bar.

Settings & theme

⌘, for System / Light / Dark appearance and a default new-image size — both persisted.

A closer look

Familiar interactions, native controls, and pixel-level precision.

Built the honest way

An HTML <canvas> drawing engine inside a Tauri native shell.

Tauri v2

Rust shell — native window, menus, file dialogs, app bundle.

React 19 + Vite

TypeScript frontend, typechecked and bundled on every PR.

Tailwind v4

CSS-first styling with light / dark theme tokens.

Zustand

UI & config state; pixels live in the canvas engine, never in React.
